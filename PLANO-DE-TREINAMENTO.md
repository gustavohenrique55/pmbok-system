## MÓDULO 1: INTRODUÇÃO AO SISTEMA (30 minutos)

### 1.1 O que é o Sistema?
- Sistema web de gestão de projetos profissional
- Funciona no navegador (Chrome, Firefox, Edge)
- Não precisa instalar nada

### 1.2 Acessando o Sistema
1. Abra seu navegador
2. Acesse: https://gustavohenrique55.github.io/Project-Management-7-PRO/
3. O sistema carrega automaticamente

### 1.3 Primeiro Contato
- Explore a tela inicial
- Identifique os menus principais
- Familiarize-se com a interface

---

## MÓDULO 2: PROJETOS (45 minutos)

### 2.1 Criar Novo Projeto
1. Clique em "Novo Projeto" ou "+"
2. Preencha:
   - Nome do projeto
   - Descrição
   - Data de início
   - Data prevista de término
   - Prioridade (Alta/Média/Baixa)
3. Clique em "Salvar"

### 2.2 Gerenciar Projetos
- **Editar**: Clique no projeto → Altere os dados → Salve
- **Arquivar**: Projetos concluídos podem ser arquivados
- **Excluir**: Cuidado! Esta ação não pode ser desfeita

### 2.3 Exercício Prático
Crie 3 projetos de exemplo:
1. "Projeto Teste A" - Prioridade Alta
2. "Projeto Teste B" - Prioridade Média
3. "Projeto Teste C" - Prioridade Baixa

---

## MÓDULO 3: TAREFAS (60 minutos)

### 3.1 Criar Tarefas
1. Selecione um projeto
2. Clique em "Nova Tarefa"
3. Preencha:
   - Título da tarefa
   - Descrição detalhada
   - Responsável
   - Prazo
   - Status (Pendente/Em Andamento/Concluída)

### 3.2 Quadro Kanban
O sistema usa o método Kanban:
- **A Fazer**: Tarefas pendentes
- **Fazendo**: Tarefas em andamento
- **Feito**: Tarefas concluídas

### 3.3 Arrastar e Soltar
- Clique na tarefa
- Arraste para a coluna desejada
- Solte para atualizar o status

### 3.4 Exercício Prático
No "Projeto Teste A", crie 5 tarefas:
1. "Tarefa 1" - Pendente
2. "Tarefa 2" - Em Andamento
3. "Tarefa 3" - Concluída
4. "Tarefa 4" - Pendente
5. "Tarefa 5" - Em Andamento

---

## MÓDULO 4: EQUIPE E MEMBROS (30 minutos)

### 4.1 Adicionar Membros
1. Acesse a seção "Equipe"
2. Clique em "Adicionar Membro"
3. Preencha:
   - Nome
   - Email
   - Cargo/Função
   - Departamento

### 4.2 Atribuir Tarefas
- Ao criar tarefa, selecione o responsável
- Cada membro vê suas tarefas atribuídas

### 4.3 Exercício Prático
Adicione 3 membros fictícios:
1. João Silva - Desenvolvedor
2. Maria Santos - Designer
3. Pedro Costa - Analista

---

## MÓDULO 5: DASHBOARD E RELATÓRIOS (30 minutos)

### 5.1 Visão Geral
O Dashboard mostra:
- Total de projetos
- Tarefas por status
- Prazos próximos
- Gráficos de progresso

### 5.2 Indicadores Importantes
- **Taxa de Conclusão**: % de tarefas concluídas
- **Tarefas Atrasadas**: Atenção especial
- **Carga de Trabalho**: Por membro

### 5.3 Exercício Prático
Analise o dashboard e identifique:
- Quantos projetos ativos?
- Quantas tarefas pendentes?
- Alguma tarefa atrasada?

## MÓDULO 6: CHATBOT DE IA (45 minutos)

### 6.1 Ativando a IA
1. Clique no ícone do chatbot (geralmente no canto)
2. A janela de chat abre
3. Digite sua pergunta

### 6.2 O que a IA pode fazer?
- Responder dúvidas sobre o sistema
- Sugerir organização de tarefas
- Ajudar com planejamento
- Dar dicas de produtividade

### 6.3 Exemplos de Perguntas
- "Como criar um novo projeto?"
- "Como mudar o status de uma tarefa?"
- "Como adicionar um membro à equipe?"
- "Qual a melhor forma de organizar minhas tarefas?"

### 6.4 Exercício Prático
Faça 5 perguntas ao chatbot:
1. "Olá, como você pode me ajudar?"
2. "Como crio uma tarefa?"
3. "O que é o método Kanban?"
4. "Como vejo relatórios?"
5. "Dê dicas de produtividade"

## DICAS IMPORTANTES

### Para Memorizar
1. **Pratique diariamente** - 15 minutos por dia
2. **Crie projetos reais** - Use para suas tarefas do dia a dia
3. **Use o chatbot** - Tire dúvidas sempre que precisar
4. **Explore os menus** - Descubra todas as funcionalidades

### Erros Comuns a Evitar
- Não salvar após editar
- Esquecer de atualizar status das tarefas
- Não definir prazos realistas
- Criar muitas tarefas de uma vez

### Atalhos Úteis
- Clique duplo para editar rapidamente
- Arraste para reorganizar
- Use filtros para encontrar itens
